# Satyam Bhanot Portfolio Website

Personal portfolio site built with HTML, CSS, and JavaScript.

Live site: [satyambhanot.github.io](https://satyambhanot.github.io)

## About

This site is my personal portfolio. It highlights my background as a Computer Science Honours student at the University of Manitoba, my work in tech support, mentoring, and a few projects that reflect how I think and work.

The project is fully static and runs without frameworks or a build step.

## Features

- Single-page portfolio with smooth section navigation
- Dark and light theme toggle
- Responsive layout for desktop and mobile
- Project, experience, and skills sections with tabbed views
- Contact form that opens a prefilled email draft
- Lightweight interactive background and scroll effects

## Tech Stack

- HTML
- CSS
- JavaScript
- Google Fonts
- GitHub Pages

## Run Locally

```bash
git clone https://github.com/satyambhanot/satyambhanot.github.io.git
cd satyambhanot.github.io
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Editing Notes
